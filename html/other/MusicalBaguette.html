Musical Baguette


This project aim to transform any kind of physical element (here a baguette) into a musical instrument. I built a home made Theremin coupled with a percussion sensor. Both sensors are into the baguette and send the informations to a computer that analyse and transform them into MIDI signal, thus can control any virtual instrument.



If you are curious you can learn how to build this on this instructable.
